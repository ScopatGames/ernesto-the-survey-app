# Ernesto the surveyor

## Installation Instructions

### To run locally:
1. Clone repository.
2. Navigate to the cloned repository.
3. ```npm install```
4. ```npm run start```
5. Open browser to localhost 8080.

### To build production version
```webpack -p```
